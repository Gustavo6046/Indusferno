Detailed Description

A distinct load of cargo in a station.

A 'load' of cargo is an amount of cargo waiting in a station, with a cargo type and a defined origin station.

Any cargo of a different type or origin must be grouped into a different 'load'. Conversely, no two loads exist with the exact same cargo type and origin.

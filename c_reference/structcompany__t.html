unsigned int company_t::chairmen[MAX_CHAIRMEN_PER_COMPANY]

The list of managing players.

Each number is the PlayerNumber of a player managing this company.

In a singleplayer game, only the first item will be set. In a multiplayer game, this must be updated manually when players join and leave.
